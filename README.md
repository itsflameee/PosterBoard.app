# Installation
1. Install [this Siri Shortcut](https://github.com/itsflameee/PosterBoard.app/releases/download/siri/ForcePosterBoardLaunch.shortcut)
2. Download [this icon](https://github.com/itsflameee/PosterBoard.app/blob/main/PosterBoard%20Icon.png?raw=true)
3. Add a shortcut to your home screen and select the name "PosterBoard" and set the previously downloaded icon.
